Dress Code

Quem nunca participou da Festa Ligo pode estar pensando qual é o dress code ou que roupas devem usar na nossa festa.
É bem simples, vista-se de maneira confortável! Se você quiser vir vestido da maneira típica como nas festas juninas do Brasil, você pode! Se você tiver trajes típicos da Letônia ou de outro país, pode usar também!

A unica regra é se vestir confortavelmente para poder aproveitar a Festa Ligo. Na Letônia durante a Festa Ligo as mulheres usam coroas de flores na cabeça e os homens coroas feitas com folhas de Carvalho. Sinta-se a vontade para usar coroas ou tiaras.
